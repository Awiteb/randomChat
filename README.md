## التنزيل

يتم استعمال [GitHub](https://github.com) لتنزيل ملف البوت

<div dir="ltr">

```bash
# clone bot form github
git clone https://github.com/Awiteb/randomChat

# cd to bot folder
cd randomChat

# install required packages
pip3 install -r requirements.txt
```
> full the config file
```bash
python3 main.py
```
